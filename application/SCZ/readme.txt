Download the SCZ summary statistics from https://walters.psycm.cf.ac.uk/; the link is under 'CLOZUK+PGC2 meta-analysis summary statistics'
Format and preprocess the summary statistics using 'format_gwas.py' and 'preprocess_gwas.R'
Run the 'run_SCZ.R' script for each weight to perform the analysis; see 'run_SCZ_analysis.sh' for how to batch these analyses and 'run_SCZ.sbatch' to see the resources required
Run the 'results_SCZ.Rmd' file to parse the results
